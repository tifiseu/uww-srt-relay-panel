UWW SRT Relay Panel

Quick reference

Each relay takes an SRT stream from KiloLink (source) and pushes/serves it to a destination (TV taker, Brightcove, Flo, EVS).

Auto mode detection

Mode is detected automatically from the URL — no dropdown needed:

URL formatModeMeaning
srt://host:portCallerConnects to remote host
srt://:portListenerOpens port, waits for connection

You can override by adding ?mode=caller or ?mode=listener to the URL manually.

Port ranges

RangePurpose
30000-30099Reserved — KiloLink SRT outputs
30100-30200KiloLink → Panel source ports
30201-30300KiloLink TURN server — DO NOT USE
30400-30499Available for listener-mode outputs (TV takers pull)
⚠ Ports 30201-30300 are used by KiloLink's built-in TURN server (coturn). Do not assign listener relays to these ports — they will conflict silently.

Stats display

Counters (pkt loss, retrans, dropped, stats pts) show total session value. After clicking Reset stats, they show total / since-reset. Color is based on the since-reset value when a reset is active.

Auto-reconnect

If the source (KiloLink) or destination drops, the relay automatically retries every 3 seconds. The dots next to URLs show connection state: ● connected, ● retrying, ● no connection. You only need to manually stop a relay — it will keep reconnecting on its own.
